# SongGraveyard 🎧🪦

> **让每一个没写完的音乐灵感拥有第二次生命。**

📖 [English README](README.md)

SongGraveyard 是一个 AI 音乐工具,把你废弃的音乐**动机**——哼唱、乐器片段、歌词碎片、
语音备忘、demo——变成一座可浏览、可关联、可复活的"灵感墓地"。

每个创作者都有一座废弃灵感的墓地。SongGraveyard 给每个灵感一个安息之处,把它和你
其他的碎片关联起来,再用 **ACE-Step 1.5** 把它复活成新的作品。

> **与 Suno 的区别:** Suno 从 prompt 生歌;SongGraveyard 从你**自己的废弃素材**出发——
> 存档、关联、复活,让散落的灵感慢慢沉淀成可发展为 EP / 概念专辑的创作资产。

## ✨ 能做什么

- **录入 / 埋葬** — 上传一段哼唱/乐器/歌词,配上图片、文字、情绪与地点。系统自动生成
  标题和**墓志铭**,把它作为一个节点葬入墓地。
- **灵感墓地** — 类 Obsidian 的关系网络。共享情绪/地点/项目(或曾被一起 remix)的碎片
  会自动连线;被你常打理的节点会变大。
- **让它复活** — 进入任意动机,选择复活方式:
  - **Resurrect** → 把片段续写成完整 demo(最忠实原旋律)
  - **Grow** → 长成完整、编曲齐全的曲子
  - **Ghost** → 一段短促、残缺、混响弥漫的空灵回声
  - **Remix** → 把两个或更多动机融合成新的作品
- **对比与保存** — 把原始与每个生成版本并排试听,下载或收藏你喜欢的。

## 🎧 气质

暗黑、诗意、梦幻——一座**音乐记忆的数字墓园**,而非恐怖屋。深蓝黑的夜、幽灵蓝与雾绿
微光,值得复活的灵感会被暖光点亮。
